About VMware Tools Help
With this help system, you can view product documentation, display context-sensitive topics, search on words and phrases, and print help topics.
The VMware Tools help system contains topics that help you use the VMware Tools control panel. These topics are not product-specific. For information about which VMware Tools features are available in the VMware product you are using, see the documentation for that VMware product.
Using This Help System
This help system uses the features of your Web browser as well as some additional capabilities to help you access product information. You can search the help using terms, phrases, and wildcards with the following limitations:
■
Searches for multiple terms retrieve only topics that include all the search terms.
■
You cannot use Boolean operators such as AND, OR, and NOT in your searches.
