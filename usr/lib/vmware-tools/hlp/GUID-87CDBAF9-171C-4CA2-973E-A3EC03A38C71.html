Show a VMware Tools Icon in a Windows Taskbar
On Windows guests, you can configure whether the VMware Tools icon appears in the notification area of the taskbar. This icon indicates whether VMware Tools is running and whether an update is available.
Prerequisites
Make sure the VMware Tools control panel is open in the guest operating system.
Procedure
1
Click the Options tab in the VMware Tools control panel.
2
Select Show VMware Tools in the taskbar.
When the guest operating system is running, if VMware Tools cannot start up, a red slash appears over the VMware Tools icon.
3
Select Notify if update is available.
When the guest operating system is running, if a VMware Tools upgrade is available, a yellow caution icon appears over the VMware Tools icon.
