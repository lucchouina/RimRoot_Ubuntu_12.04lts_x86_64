NAME

evs_finalize − Terminate a connection to the EVS service

SYNOPSIS

#include <corosync/evs.h>

int evs_finalize(evs_handle_t handle);

DESCRIPTION

The evs_finalize function is used to close a connection to the extended virtual synchrony API. Once the connection is finalized, the handle may not be used again by applications. No more callbacks will be dispatched from the evs_dispatch function.

RETURN VALUE

This call returns the EVS_OK value if successful, otherwise an error is returned.

ERRORS

The errors are undocumented.
