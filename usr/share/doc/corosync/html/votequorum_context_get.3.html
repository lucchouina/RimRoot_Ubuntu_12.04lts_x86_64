NAME

votequorum_context_get − Gets the context variable for a VOTEQUORUM instance

SYNOPSIS

#include <corosync/votequorum.h>

int votequorum_context_get(votequorum_handle_t handle, void **context);

DESCRIPTION

The votequorum_context_get function is used to retrieve the context variable previously stored using votequorum_context_set(3)

RETURN VALUE

This call returns the VOTEQUORUM_OK value if successful, otherwise an error is returned.

ERRORS

The errors are undocumented.
